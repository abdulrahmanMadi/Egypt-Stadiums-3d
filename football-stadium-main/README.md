# StadiView

Interactive procedural 3D football stadium seat preview (Angular 22).

## Try it locally

```bash
npm install
npm start
```

Then open the local address shown by the Angular CLI (usually `http://localhost:4200`).

## Controls

| Action                     | Control                                       |
| -------------------------- | --------------------------------------------- |
| Rotate around the stadium  | Click or touch and drag                       |
| Zoom                       | Mouse wheel or on screen controls             |
| Preview a seat             | Click a seat                                  |
| Look around from a seat    | Drag while in seat view                       |
| Leave seat view            | Press Esc or use “Back to stadium”            |
| Confirm the demo selection | Press Enter or use “Grab seat”                |

## Notes

- The original single-file demo is kept as `index.html` / `stadium-original.html`.
- Previous Vite starter files are under `legacy-vite/`.
- The live app runs from `src/` via Angular.
