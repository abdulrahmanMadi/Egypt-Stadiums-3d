# StadiView — Football Stadium Experience

This repository contains an Angular and Three.js interactive 3D football
stadium experience maintained by
[Abdulrahman Madi](https://github.com/abdulrahmanMadi).

The application source is in [`football-stadium-main/`](football-stadium-main/).
See the [full project documentation](football-stadium-main/README.md) for
features, screenshots, controls, setup instructions, attribution, and licensing.

## Quick start

```bash
git clone https://github.com/abdulrahmanMadi/football-stadium-main.git
cd football-stadium-main/football-stadium-main
npm install
npm start
```
